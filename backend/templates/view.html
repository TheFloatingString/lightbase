Lightbase.

Lightbase is a P2P database library, which is:

  • Affordable. Free as long as possible.
  • Frictionless. Fast and easy for developers to use. Built on REST API (minimal depencencies require) and HTTP-inspired commands.
  • Uniqueness-first. Creates unique items that can be accessed across different databases.

Future features include:

  • Better item retrieval visibility for users and developers (ie keeping track of hashes)
  • Interoperability between different P2P databases
